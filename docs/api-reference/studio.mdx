## Overview

Studio is DataLaser's computational environment. It provides sandboxed Python execution against file and database sources, a full suite of statistical analysis endpoints, AI-powered code generation, and notebook management. All execution happens server-side in isolated containers.

---

## Execute Code (File Source)

Execute Python code against a file-based data source in a sandboxed environment. The source data is available as a pandas DataFrame named `df`.

```
POST /api/studio/execute
```

### Request

<ParamField body="source_id" type="string" required>
  The file-based data source UUID.
</ParamField>

<ParamField body="code" type="string" required>
  Python code to execute. The source data is pre-loaded as `df` (a pandas DataFrame). Use `result` to return output.
</ParamField>

<ParamField body="timeout_seconds" type="integer">
  Execution timeout. Defaults to `30`. Maximum `120`.
</ParamField>

### Response

```json
{
  "status": "success",
  "execution_time_ms": 245,
  "result": {
    "type": "dataframe",
    "columns": ["region", "total_revenue", "order_count"],
    "rows": [
      ["DACH", 18500000, 4500],
      ["Nordics", 12800000, 3200],
      ["Benelux", 9600000, 2800]
    ],
    "shape": [3, 3]
  },
  "stdout": "",
  "stderr": "",
  "charts": []
}
```

### Example

```bash
curl -X POST https://datalaser.de/api/studio/execute \
  -H "Authorization: Bearer $DATALASER_API_KEY" \
  -H "X-Project-Id: proj_abc123" \
  -H "Content-Type: application/json" \
  -d '{
    "source_id": "src_8f3a1b2c",
    "code": "result = df.groupby(\"region\").agg({\"revenue\": \"sum\", \"order_id\": \"count\"}).rename(columns={\"revenue\": \"total_revenue\", \"order_id\": \"order_count\"}).reset_index()",
    "timeout_seconds": 30
  }'
```

### Errors

| Status | Code                 | Description                              |
|--------|----------------------|------------------------------------------|
| `400`  | `SYNTAX_ERROR`       | Python code has a syntax error.          |
| `404`  | `SOURCE_NOT_FOUND`   | Source ID does not exist.                |
| `408`  | `EXECUTION_TIMEOUT`  | Code exceeded the timeout limit.         |
| `422`  | `RUNTIME_ERROR`      | Code raised an unhandled exception.      |
| `422`  | `SANDBOX_VIOLATION`  | Code attempted a disallowed operation.   |

## Execute Code (Database Source)

Execute Python code against a live database connection. The database connection is available via `conn` (SQLAlchemy engine) and you can use `pd.read_sql()`.

```
POST /api/studio/execute-db
```

### Request

<ParamField body="source_id" type="string" required>
  The database source UUID.
</ParamField>

<ParamField body="code" type="string" required>
  Python code to execute. A SQLAlchemy `conn` object and `pd` (pandas) are pre-imported. Use `result` to return output.
</ParamField>

<ParamField body="timeout_seconds" type="integer">
  Execution timeout. Defaults to `30`. Maximum `120`.
</ParamField>

### Response

Same shape as the [Execute Code](#execute-code-file-source) response.

### Example

```bash
curl -X POST https://datalaser.de/api/studio/execute-db \
  -H "Authorization: Bearer $DATALASER_API_KEY" \
  -H "X-Project-Id: proj_abc123" \
  -H "Content-Type: application/json" \
  -d '{
    "source_id": "src_d4e5f6a7",
    "code": "df = pd.read_sql(\"SELECT region, SUM(revenue) as total FROM orders GROUP BY region\", conn)\nresult = df"
  }'
```

### Errors

Same error codes as [Execute Code](#execute-code-file-source), plus:

| Status | Code                  | Description                              |
|--------|-----------------------|------------------------------------------|
| `422`  | `CONNECTION_FAILED`   | Could not connect to the database.       |

## Suggest Code

Generate Python code cells from a natural language description. Powered by AI with context from the source schema.

```
POST /api/studio/suggest
```

### Request

<ParamField body="source_id" type="string" required>
  The data source UUID.
</ParamField>

<ParamField body="prompt" type="string" required>
  Natural language description of what you want to do, e.g. `"Show revenue by region as a bar chart"`.
</ParamField>

<ParamField body="context" type="array">
  Array of previous code cells for conversational context.
</ParamField>

### Response

```json
{
  "cells": [
    {
      "type": "code",
      "language": "python",
      "source": "import matplotlib.pyplot as plt\n\nrevenue_by_region = df.groupby('region')['revenue'].sum().sort_values(ascending=False)\n\nfig, ax = plt.subplots(figsize=(10, 6))\nrevenue_by_region.plot(kind='bar', ax=ax, color='#191919')\nax.set_title('Revenue by Region')\nax.set_ylabel('Total Revenue')\nax.set_xlabel('Region')\nplt.tight_layout()\nresult = fig",
      "explanation": "Groups the data by region, sums revenue, and creates a bar chart sorted by total revenue descending."
    }
  ]
}
```

### Example

```bash
curl -X POST https://datalaser.de/api/studio/suggest \
  -H "Authorization: Bearer $DATALASER_API_KEY" \
  -H "X-Project-Id: proj_abc123" \
  -H "Content-Type: application/json" \
  -d '{
    "source_id": "src_8f3a1b2c",
    "prompt": "Show revenue by region as a bar chart"
  }'
```

### Errors

| Status | Code                | Description                        |
|--------|---------------------|------------------------------------|
| `404`  | `SOURCE_NOT_FOUND`  | Source ID does not exist.          |
| `422`  | `GENERATION_ERROR`  | Code generation failed.            |

## Notebooks CRUD

Manage Studio notebooks (create, list, read, update, delete).

### Create Notebook

```
POST /api/studio/notebooks
```

<ParamField body="name" type="string" required>
  Notebook display name.
</ParamField>

<ParamField body="source_id" type="string" required>
  The data source this notebook is associated with.
</ParamField>

<ParamField body="cells" type="array">
  Initial array of notebook cells. Defaults to an empty notebook.
</ParamField>

**Response:**

```json
{
  "id": "nb_f1g2h3i4",
  "name": "Revenue Analysis",
  "source_id": "src_8f3a1b2c",
  "cells": [],
  "created_at": "2026-03-24T11:00:00Z",
  "updated_at": "2026-03-24T11:00:00Z"
}
```

### List Notebooks

```
GET /api/studio/notebooks
```

Returns all notebooks for the current project (set via `X-Project-Id` header).

### Get Notebook

```
GET /api/studio/notebooks/{notebook_id}
```

### Update Notebook

```
PATCH /api/studio/notebooks/{notebook_id}
```

<ParamField body="name" type="string">
  Updated name.
</ParamField>

<ParamField body="cells" type="array">
  Updated cell array. Replaces all cells.
</ParamField>

### Delete Notebook

```
DELETE /api/studio/notebooks/{notebook_id}
```

### Example

# Create a notebook
